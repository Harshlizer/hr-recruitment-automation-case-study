# Mock Planner Task

## Title

Security Review - Alex Example - Operations Specialist

## Description

Candidate intake record created automatically from Microsoft Forms submission.

Fields included in task details:

- Position
- Hiring manager
- Employment country
- Work mode
- Start date
- Candidate contact details
- Consent flags
- Prior employment summary
- Additional comments

## Outcome Convention

- `Approved`: Candidate can proceed to HR next step
- `Rejected`: Candidate cannot proceed
- `Needs clarification`: HR follow-up required
